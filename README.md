# 项目简介
## 核心功能
- 管理员功能：用户管理、问卷题目管理、意见反馈管理、问卷调查管理、管理员管理、系统管理、个人信息管理
- 用户功能：个人中心、修改密码、答卷记录管理、我的收藏管理

## 适用场景
- 教育机构
- 企业调查
- 市场调研
- 社会调查

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
